# Addie's Developer Hub

Welcome to **Addie's Developer Hub**, my personal portfolio and learning laboratory where I document my journey as an educator, designer, and front-end developer.

This website showcases the projects, skills, and experiences that have shaped my growth while serving as a space to continually practice and refine my web development abilities.

---

## About the Project

The Developer Hub was built entirely from scratch using HTML, CSS, and JavaScript. Rather than relying on templates or website builders, every page has been designed and developed as an opportunity to strengthen my understanding of front-end development, responsive design, and user experience.

Each featured project is presented as an **Expedition**, highlighting not only the final product but also the design process, challenges, and lessons learned along the way.
